# 排任务工具

## 功能介绍

这是一个在线排任务工具，帮助您减轻排任务工作量，提高排任务效率。主要功能包括：

### 1. 任务池管理
- 添加任务到任务池，包含任务名称、任务节点、所需技能、所需人力
- 查看任务列表，显示任务状态
- 删除任务

### 2. 人员管理
- 添加人员
- 为人员添加技能
- 标记人员请假状态
- 删除人员

### 3. 排任务逻辑
- 人员没排满8h需补齐8H，并展示出哪些人没有排满（空余人员）
- 任务没有排需罗列清楚，哪些任务是没有排的
- 排任务优先按照人员能力、任务节点按照时间优先级
- 请假人员无需排任务

### 4. 排任务结果
- 查看已排任务列表
- 查看未排任务列表
- 查看空余人员列表

## 使用方法

1. 打开 `index.html` 文件即可使用
2. 在「任务池」标签页添加任务
3. 在「人员管理」标签页添加人员和技能
4. 在「排任务」标签页设置排任务日期并开始排任务
5. 在「排任务结果」标签页查看排任务结果

## 技术实现

- 前端：HTML5 + CSS3 + JavaScript
- 数据存储：浏览器内存（刷新页面后数据会丢失）
- 响应式设计：适配不同屏幕尺寸

## 注意事项

- 本工具仅在浏览器本地运行，数据不会保存到服务器
- 刷新页面后，所有数据会重置
- 建议在排任务完成后，截图保存排任务结果
